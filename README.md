# ☕ Coffee Shop Sales Dashboard

An interactive **Excel Dashboard** project built to analyze coffee shop sales performance, customer behavior, and product trends using data visualization and business analytics techniques.

## 📌 Project Overview

This project focuses on transforming raw coffee shop sales data into meaningful business insights through an interactive Excel dashboard.
The dashboard helps analyze:

* Overall sales performance
* Customer footfall
* Product category contribution
* Peak sales hours
* Store-wise performance
* Order trends across weekdays
* Top-selling products

The goal of this project is to support better business decision-making through data-driven insights.

## 🚀 Key Features

### 📊 Sales & Business Metrics

* **Total Sales:** $98,330.31
* **Total Footfall:** 21,096
* **Average Bill per Person:** 4.66
* **Average Orders per Person:** 1.43

* Analyzed quantity ordered across different hours

## 📈 Business Insights Generated

* Morning hours showed the highest sales volume
* Coffee contributed the largest percentage of total sales
* Regular and Large size drinks were most preferred
* Hell’s Kitchen store generated the highest sales
* Barista Espresso was the top-selling product
